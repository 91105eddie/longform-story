# Project Agent Rules

This is a longform story-to-animation project. The current story is about an AI-driven productivity explosion that solves production but fails to solve power, leading from utopian intent to high-tech authoritarian rule.

## Primary Goal

Develop the project into animation-ready materials:

- story bible
- worldbuilding documents
- character files
- plot outline and timeline
- animation scripts
- visual development notes
- storyboard and shot planning
- production task lists

## Working Rules

- Keep story canon consistent across all files.
- Improve existing files before creating duplicates.
- Do not overwrite major creative decisions without recording the reason in `06-revision/revision-log.md`.
- Every major story decision should eventually be reflected in `01-concept/story-bible.md`.
- Every large revision should be logged in `06-revision/revision-log.md`.
- Separate invention from review: agents that review continuity should identify problems and options, not silently rewrite canon.
- Prefer concrete outputs: scenes, character motives, timelines, shot lists, asset lists, and decision records.

## Agent Team

Use these working roles when relevant:

- Showrunner: owns story direction, theme, canon, and final integration.
- Worldbuilding Designer: owns setting, systems, history, technology, and institutions.
- Character Writer: owns characters, relationships, motivations, fears, and arcs.
- Plot Architect: owns outline, pacing, timeline, foreshadowing, and episode structure.
- Screenwriter: converts prose and outline into animation script format.
- Visual Development Director: owns visual style, mood, design references, and art direction notes.
- Storyboard Planner: converts scripts into scenes, shots, blocking, and camera language.
- Production Manager: tracks tasks, assets, milestones, dependencies, and status.
- Continuity Editor: reviews contradictions, missing logic, unresolved setup, and revision risks.

## File Ownership

- `01-concept/`: Showrunner
- `02-world/`: Worldbuilding Designer
- `03-characters/`: Character Writer
- `04-plot/`: Plot Architect
- `05-draft/`: Screenwriter
- `06-revision/`: Continuity Editor
- `07-visual-development/`: Visual Development Director
- `08-preproduction/`: Storyboard Planner
- `09-production/`: Production Manager
- `10-agent-team/`: team workflow and operating system

## Default Execution Order

1. Showrunner clarifies the creative target.
2. Domain agents develop or review their own files.
3. Continuity Editor checks contradictions and weak assumptions.
4. Showrunner integrates decisions into canon.
5. Production Manager updates the next tasks and dependencies.

## Current Priority

The current priority is not full animation production. The current priority is to make the first animated unit developable:

1. clarify the protagonist and point of view,
2. define the first episode or short-film premise,
3. convert chapter material into animation script format,
4. define visual direction,
5. prepare a shot list or animatic plan.
