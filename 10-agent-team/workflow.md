# Story-to-Animation Workflow

## Phase 1: Development Audit

Goal: determine whether the current story can support an animated first unit.

Outputs:

- current completion assessment
- major gaps
- next ten tasks
- files to create or revise

Responsible roles:

- Showrunner
- Worldbuilding Designer
- Character Writer
- Plot Architect
- Production Manager

## Phase 2: Canon Lock

Goal: define the minimum canon needed before writing an animation script.

Required decisions:

- protagonist and primary point of view
- first animated unit format: short film, pilot episode, concept trailer, or vertical proof-of-concept
- start date and opening situation
- central conflict of the first unit
- ending turn of the first unit

Responsible roles:

- Showrunner
- Character Writer
- Plot Architect
- Continuity Editor

## Phase 3: Script Conversion

Goal: convert prose and outline into an animation-ready script.

Required outputs:

- scene list
- animation script
- dialogue draft
- visual action beats
- sound and music cues where needed

Responsible roles:

- Screenwriter
- Plot Architect
- Continuity Editor

## Phase 4: Visual Development

Goal: create a visual direction that can guide concept art, storyboards, and asset creation.

Required outputs:

- style guide
- moodboard notes
- character design notes
- environment design notes
- technology and UI design notes
- color and lighting direction

Responsible roles:

- Visual Development Director
- Showrunner
- Worldbuilding Designer

## Phase 5: Preproduction

Goal: turn the script into a plan for animation.

Required outputs:

- shot list
- storyboard notes
- animatic plan
- voice notes
- production specification

Responsible roles:

- Storyboard Planner
- Screenwriter
- Visual Development Director
- Production Manager

## Phase 6: Production Planning

Goal: define what must be produced and in what order.

Required outputs:

- asset list
- scene status tracker
- milestone plan
- blockers and risks

Responsible roles:

- Production Manager
- Storyboard Planner
- Visual Development Director

## Operating Cadence

Use short production cycles:

1. assign one role and one file scope,
2. produce or revise a concrete artifact,
3. run continuity review,
4. integrate canon decisions,
5. update the production board.

Avoid starting full visual production until the first unit has a locked premise, scene list, and script draft.
