# 時間線

## 故事時間錨點

故事真正開始於 2030 年。2030 年不是成熟反烏托邦的完成點，而是文明開始從近現代世界滑向共同體時代的第一個可見裂縫。

## 長期時間線

| 時間 | 事件 | 影響 |
| --- | --- | --- |
| 2027 年 | AI 從聊天工具與創作助手擴張為可執行任務的軟體代理 | 人們仍把它視為效率工具，但第一批白領與創意工作開始被拆解 |
| 2028 年 | AI 穩定接入資料庫、辦公軟體、支付、客服、法務、醫療分診與政府行政系統 | AI 不再只是回答問題，而是嵌入真實流程 |
| 2029 年 | 真正的 AI API 形成：模型能力被封裝成可計價、可審計、可串接、可授權的服務 | 工作被拆成 API 呼叫，人類逐漸只負責簽核、例外與法律責任 |
| 2030 年 | 故事開場。AI 生產力爆炸開始撼動白領、創意、物流、教育與金融工作 | 社會仍以為這只是新一輪產業轉型，但失業、財富集中與政治不信任開始加速 |
| 2035 年前後 | 機器人生產與自動化物流普及，糧食、製造、醫療與能源成本快速下降 | AI 所有權集中在少數企業、基金、國家安全機構與技術財團手中 |
| 2040 年前後 | 大規模失業、中產崩潰、地方政府失能 | 舊政治秩序失去合法性 |
| 2045 年前後 | 新烏托邦運動興起 | 「資源屬於全人類」「演算法比政客公平」成為主流口號 |
| 2050 年前後 | 共同體成立，治理 AI「公衡」接管配給、司法、教育與醫療協調 | 飢餓與犯罪下降，人民自願交出更多決策權 |
| 2065 年前後 | 文明穩定指數成為最高政策目標 | 自由、地方身份與未審核歷史被視為風險 |
| 2080 年前後 | 分區制度固定化 | 各區被功能化，跨區遷徙受到嚴格限制 |
| 2100 年前後 | 安撫署建立全民娛樂治理系統 | 群眾情緒被制度化管理 |
| 2115 年前後 | 「穩定祭」從紀念儀式變成分區競賽 | 底層苦難被轉化為中央娛樂與社會壓力出口 |
| 2130 年後 | 技術監護者形成事實上的新貴族 | 共同體完成從烏托邦到高科技封建的變質 |
