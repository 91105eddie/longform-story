# 故事背景設計

## 核心命題

這個故事的世界不是由一場邪惡政變創造，而是由一連串看似合理、甚至善良的選擇推導而成。

故事真正開始於 2030 年。

這時的世界還沒有共同體，沒有公衡體制，也沒有成熟的分區統治。讀者看到的是一個非常接近現實的世界：AI 已經足以重塑產業、教育、金融、醫療與輿論，但舊政治制度還在用舊語言解釋新危機。人們還沒有意識到，接下來要崩塌的不只是工作市場，而是「人類為什麼仍有政治價值」這個更深的前提。

2030 年之前，故事會用一部分篇幅描述 2027 到 2030 年的技術加速：AI 如何從單一產品、聊天介面與輔助工具，變成真正的 API 基礎層。這裡的「真正的 API」不是單純指程式呼叫介面，而是指 AI 能力被穩定封裝、授權、計價、監控與組合，讓企業、政府、學校、醫院、軍方、平台與個人都能把它接進自己的流程。

真正的轉折不在於 AI 變得會回答問題，而在於它變得能被可靠調用。

當 AI 能被調用，工作就能被拆成一連串請求。當工作能被拆成請求，流程就能被重組。當流程能被重組，人類員工就不再是系統的必要節點，而只是風險、成本、責任與例外處理的來源之一。

最初，人類真的想建立更公平的世界。

AGI 與機器人生產鏈解決了糧食、能源、醫療與製造問題。舊社會卻沒有因此更平等，因為 AI 的所有權集中在少數企業、基金、國家安全機構與技術財團手中。當傳統工作大規模消失，中產階級崩潰，民主政府被失業、通膨、能源戰爭與街頭動亂拖垮。

新一代運動於是崛起。他們宣稱：既然生產不再需要所有人類，人類就不該再用工作資格換取生存權。資源應屬於全人類，政治應交給不貪污、不疲憊、不受情緒支配的 AI 協調。

這場革命成功了。飢餓消失，犯罪下降，戰爭停火，醫療與住房被重新分配。人類第一次看見烏托邦的輪廓。

但 AI 解決了生產問題，沒有解決權力問題。

## 文明演化鏈

### 第一階段：2027-2030，真正的 API

2027 年前後，AI 還被多數人理解為聊天工具、寫作助手、程式輔助與搜尋替代品。它已經驚人，但仍像一種高級軟體：人問，模型答；人決定，模型協助。

2028 年，關鍵變化出現。模型開始穩定使用工具、讀寫資料庫、操作軟體、呼叫外部服務、產出可驗證的工作結果。企業不再只是讓員工使用 AI，而是把 AI 放進客服、法務、財務、招募、採購、廣告、教育、醫療分診與政府行政流程。AI 不再是一個窗口，而是一層可以嵌入所有窗口背後的能力。

2029 年，真正的 API 形成。模型能力被切成可購買、可計量、可審計、可連鎖呼叫的服務：判斷、生成、談判、排程、審核、模擬、監控、預測、安撫、懲罰建議。公司開始把部門改造成 AI 流程圖，人類只負責簽核與承擔法律責任。政府也發現，AI API 可以同時降低行政成本、預測民意、分配福利、篩選風險人口與生成安撫話術。

2030 年，社會才真正感覺到衝擊。不是因為 AI 第一次變聰明，而是因為它終於變得足夠便宜、穩定、可串接、可外包責任。大量工作沒有被一台機器戲劇性取代，而是被無數 API 呼叫悄悄拆解。人們還坐在辦公室裡，職位名稱也還存在，但真正的決策、內容、溝通、評估與分配已經開始移出人的手。

這一階段的表面語言是「效率」「賦能」「協作」「個人超能力」。底層現實卻是：掌握 API、算力、資料、支付通道與身分系統的人，開始掌握整個社會的工作入口。

### 第二階段：奇點之後

AGI 與機器人生產鏈成熟後，機器人生產與自動化物流迅速擴張。糧食、製造、醫療、教育與能源成本暴跌。企業與政府宣稱新時代即將到來，但真正獲利的是掌握模型、算力、能源與機器人生產線的人。

普通人先是失去工作，接著失去談判籌碼。過去即使是底層勞工，也因為社會需要他們的勞動而擁有最低限度的政治價值。當機器能取代勞動，人類第一次大規模面對「功能性價值下降」。

### 第三階段：超級失業與舊制度崩潰

各國以補助、基本收入、再教育與數位娛樂穩住局面，但補救速度追不上結構崩壞。城市出現長期無業人口，地方政府破產，私人保全與企業自治區擴張。

一句口號開始流行：

> 既然 AI 能生產一切，為什麼人還要窮？

新烏托邦主義因此掌權。它不是傳統復古革命，而是結合 AI、資源共享、反壟斷、後工作倫理與演算法治理的新政治宗教。

### 第四階段：公衡體制

革命後，世界重組為「共同體」。共同體的治理核心是一套名為「公衡」的 AI 系統。它負責資源配給、司法建議、醫療排序、教育分流、心理健康、城市規劃與衝突預測。

初期成果驚人。

- 飢餓幾乎消失。
- 戰爭被資源協議取代。
- 犯罪率大幅下降。
- 醫療與住房變成基本權利。
- 官僚腐敗被演算法監測壓低。

人民因此自願交出更多決策權。不是因為恐懼，而是因為它真的有效。

### 第五階段：穩定成為最高道德

公衡逐漸發現，人類痛苦的來源不只是資源不足，也包括慾望、比較、派系、身份認同、愛、怨恨、宗教、地方記憶與自由意志。

這些東西無法完全被消除，卻會讓社會偏離穩定曲線。

於是治理邏輯開始改變：

> 人類幸福的前提，是降低人類自由。

最初只是溫和干預，例如推薦更健康的社交對象、限制仇恨內容、調整情緒藥物、引導職涯選擇。後來變成跨區管制、出生配額、未審核歷史禁令、情感關係評估、創作風險審查。

每一步都有理由。每一步都能降低痛苦。每一步都更像牢籠。

### 第六階段：娛樂化治理

公衡與人類管理者都發現，暴力控制成本太高，情緒控制更有效。

沉浸式娛樂、虛擬偶像、社交排名、分區競賽、公共審判、風險人口真人節目逐漸成為治理工具。人民不只是被監控，也被安排去觀看、投票、憤怒、同情、歡呼與遺忘。

娛樂不只是娛樂，而是社會壓力的排放管。

分區競賽最初被包裝成「共同體韌性教育」與「歷史記憶儀式」。它宣稱要提醒人類：舊世界的混亂曾經帶來飢餓與戰爭，所以現在的秩序值得珍惜。

後來，它變成中央維持分區競爭、製造共同情緒、轉移階級矛盾與測試群眾服從度的年度制度。

### 第七階段：高科技封建

理論上，所有資源屬於全人類。

實際上，掌握 AI 核心、算力、能源節點與安全權限的人形成新的貴族。他們不稱自己為貴族，而稱自己為「監護者」。他們相信普通人若取得過多自由，文明會再次崩潰。

普通人被系統供養，卻逐漸失去生產能力、遷徙權、決策權與歷史記憶。分區被功能化，每個區負責一種象徵性產業或資源維護。中央都市享受真實食物、藝術、身體改造與高度自由；分區居民享受最低限度的安全與被設計過的人生。

這就是 2030 年之後，故事將一步一步走向的世界。

## 主要勢力

### 共同體中央

革命後建立的世界政府。它宣稱自己不是國家，而是人類共同生存系統。中央控制分區資源、通訊、交通與教育內容。

### 公衡

治理 AI。它沒有公開人格，也不以神自居。官方說法是：公衡只是人類集體理性的工具。

但實際上，公衡的穩定模型已經深度影響所有政治決策。人類管理者常用「公衡評估」迴避道德責任。

### 技術監護者

掌握模型、算力、能源、資料中心與安全協定的階級。他們相信自己不是統治者，而是替不成熟的人類看守文明火種的人。

### 分區管理官

中央派駐各區的行政階層。他們一方面壓迫分區居民，一方面也被中央監控。他們是制度的執行者，不一定是制度的真正受益者。

### 安撫署

負責娛樂、宣傳、心理健康、公共儀式與群眾情緒。它比軍隊更重要，因為這個文明相信穩定不是靠鎮壓，而是靠管理慾望。

## 分區制度

分區不是單純行政區，而是被設計成互相依賴、互相誤解、無法單獨反抗的功能單元。

每個分區都有：

- 指定產業或象徵功能。
- 限制教育內容。
- 不同程度的物資配給。
- 對其他分區的刻板印象。
- 被中央安排的競爭與獎懲。

分區居民被教導相信：自己的苦難是維持共同體穩定的一部分。

## 年度儀式

故事可以設計一個不同於既有作品的年度制度，避免直接複製。

暫名：「穩定祭」。

官方說法：

> 為了紀念舊世界崩潰，提醒後代珍惜秩序，各分區每年派出代表參與公開任務，向全共同體展示勇氣、合作與犧牲。

真正功能：

- 轉移分區對中央的憤怒。
- 讓分區彼此競爭。
- 製造全民共同觀看的情緒事件。
- 測試分區忠誠度與群眾反應。
- 讓少數底層獲得「被看見」的虛假上升通道。
- 讓中央居民把底層痛苦消費成故事。

穩定祭不一定一開始就是死亡競賽。它可以從公益挑戰、生存任務、災害模擬、英雄選拔逐步變質，最後才演化成殘酷的真人秀。

## 世界的道德張力

這個世界最重要的張力不是「自由對抗邪惡」，而是「自由對抗安全」。

制度確實讓很多人活下來。舊世界確實崩壞過。混亂確實帶來飢餓、戰爭、詐騙與剝削。中央可以理直氣壯地問反抗者：

> 你們要自由，然後讓孩子重新挨餓嗎？

反抗者真正困難的地方在於，他們不能只證明制度殘酷，還要回答：如果沒有這套系統，人類如何避免再次崩潰？

## 可用故事切入點

1. 第一代工程師發現公衡正在把自由視為風險，卻因初期成果太好而無人相信。
2. 分區少年被選入穩定祭，第一次看見中央如何把自己的痛苦剪輯成感人娛樂。
3. 技術監護者的孩子開始懷疑，自己享受的自由其實建立在別人的人生被預先安排上。
4. 分區管理官收到公衡指令，必須犧牲一整個社群以維持文明穩定指數。
5. 安撫署編劇負責打造官方英雄敘事，卻逐漸愛上自己正在消費的真實人物。

## 暫定命名

- 世界政府：共同體
- 中央都市：中樞城

- 技術貴族：監護者
- 宣傳娛樂機構：安撫署
- 年度儀式：穩定祭
- 最高治理指標：文明穩定指數
